Python 现在是美国顶尖大学最受欢迎的入门教学语言

由菲利普郭
2014
年7月7日 评论 （12）

总结

在撰写本文时（2014 年 7 月），Python目前是在美国顶级部门教授计算机科学入门课程的最流行的语言。

具体来说，前 10 个 CS 部门中有 8 个（80%）和前 39 个部门中的 27 个（69%）教授 Python 入门 CS0 或 CS1 课程。

动机

作为教授新手的第一语言，Python越来越受欢迎。三年前，马克·古兹迪亚在博客上谈到Python作为教学语言和对未来教学语言的预测。麻省理工学院和加州大学伯克利分校排名靠前的CS系最近将入门课程改为Python。最大的三个 MOOC 提供商（edX、Coursera 和 Udacity）都在 Python 中提供入门编程课程。不同子领域的教授（如洛雷娜·巴巴）现在提倡向新手教授Python。

就个人而言，我过去四年来一直在开发的教育工具"在线Python导师"（www.pythontutor.com），由于语言的日益普及，其使用率急剧上升。

然而，尽管有许多趣闻，我没有看到任何最近的数字量化Python在教学编程方面是多么受欢迎。因此，我试图通过记录美国顶尖计算机科学部门中有多少使用它来教授他们的入门课程来找出答案。一路上，我也结束了其他流行的教学语言的编目。

为什么进行此分析？因为选择什么语言首先反映了每个系的教学哲学，影响了许多学生对计算机科学的第一印象。美国顶尖部门选择的语言可以表明计算机科学教育的更广泛趋势，因为这些语言往往是其他教育界的潮流。

方法

我考虑前39个部门，按美国新闻在2014年排名。（随着美国新闻发布新的排名，此链接可能会过时。为什么我在39点停下来？因为有8路领带40，所以区分信号是弱的这一点。没有排名方案是完美的，排名会随时间而变化。特别是，这个排名只考虑美国授予博士学位的大学。我39年的截止排除了许多我尊重的大学，包括我自己的大学。我们可以无休止地讨论哪些学校包括，以及如何排名他们。但我不得不选择一些排名方案和截止，这感觉是合理的。

每所大学，我都寻找CS0和CS1课程在CS，CSE，或EECS系，以计算机科学的家。我定义CS1课程是 CS 专业的第一道必修课程，将CS0课程定义为非专业课程或 CS1 之前需要额外准备的人员的入门编程（而不仅仅是基本的计算机素养）。CS0 和 CS1 都不应具有任何 CS 先决条件，否则它们不能成为学生学习的第一个 CS 类。（具体来说，CS0不应成为 CS1 的先决条件。我没有计算迷你课程、特别加速 CS1+ 课程、高编号选修课或其他部门提供的编程课程。请注意，CS0 或 CS1 没有严格的官方定义，因此我不得不行使一些个人判断。我试图遵守尽可能多的每个部门的官方课程说明和先决条件表。

虽然一些教师认为入门编程意味着CS1，但为了这个分析，我特意将CS0和CS1组合在一起，因为这些课程是许多学生，无论专业还是最终职业，在大学里学习他们的第一门编程语言的地方。鉴于美国K-12计算机科学教育的不幸状况，这些大学课程是许多学生，尤其是女性和代表性不足的少数民族，可能看到他们的第一种编程语言。即使学生以编程经验到达，他们在大学里首次被正式批准的曝光仍然具有影响力。我想让CS0与CS1平起平坐，因为随着编程在科学、工程甚至非STEM学科中越来越普遍，越来越多的非专业人员正在服用CS0。此外，少数 CS 专业在 CS1 之前采用 CS0，尤其是没有过多之前编程机会的学生。因此，CS0 课程是计算的重要门户，适合那些可能永远不会知道他们有这种激情的学生。

结果

下图显示了前 39 个部门中有多少部门使用七种最常见的语言教授 CS0 或 CS1。酒吧高度加起来超过39，因为许多学校提供CS0和CS1。

讨论

即使我是Python的忠实支持者，我也会尽力不加编辑地陈述事实。

Python 是此列表中最流行的语言。它以微弱优势超过了Java，它在过去十年中一直是主要的入门教学语言。有些学校已经完全切换到Python，而另一些学校则采用混合方法，在CS0中提供Python，并将Java保留在CS1中。然而，在高中阶段，Java仍然用于AP（高级安置）课程。

下一个最流行的语言是MATLAB，它常用于CS0课程，向科学家和工程师介绍编程。C和C++是名单上的下一个，但在过去的十年里，它们已经被Java牢牢地取代。2003年，高中AP课程甚至用Java取代了C++。此外，一些使用C（如哈佛的CS50）的入门课程与其他语言一起教授，而不是让它成为唯一的语言。

基于计划的语言在教育者和编程语言研究人员中很受欢迎。最值得注意的是，两个（有点对立的）哲学阵营——SICP和HtDP——围绕计划生态系统创作了广受好评的教科书和课程。但近年来，在麻省理工学院和加州大学伯克利分校等地，计划已经逐步取消，转而支持Python。这个名单上只有五所学校使用它。

划痕是唯一使此列表成为基于块的可视语言。它是这个流派最流行的语言之一，其中包括相关的项目，如爱丽丝，应用程序发明家，艾玩具，科杜，StarLogo，和Touch发展。这类语言的创作者主要关注K-12教育，这或许可以解释为什么他们在大学一级没有得到那么多的采用。

最后，请注意，三组有趣的语言没有在此图表上使用，因为它们在我们的示例中在零或一所大学使用：

  • 静态类型功能语言，如哈斯克尔和ML变体，在PL研究人员中很受欢迎
  • 动态类型语言，如 JavaScript、Ruby 和 PHP，在 Web 程序员中很受欢迎
  • 广泛使用的行业语言，通常与特定专有平台（如 Objective-C（苹果）和 C#/Visual Basic（微软）相关联

如果我们在五年、十年或二十年内重新审视此分析，那么哪种语言将处于领先地位？

高级语言的设计有两个目的：1） 以更高效、更准确、更低成本完成工作;2） 通过语言，体系结构对程序员可见。点 （1） 具有商业方面，虽然 （2） 在运行中可能具有商业方面，但更重要的是：它帮助程序员在计算机内部工作的方式、如何排列内存、如何获取和释放内存、分配位置（cpu 或 RAM 中）、数据结构与物理内存的关系、实际执行低级操作的方式，以及算法的实际工作方式，等等。

该功能 （1） 在 C++、Java、Fortran、Perl、Python 等中很普遍，而功能 （2） 以最美丽的方式在 C 中流行，而在其它方面没有其它方面。因此，在我看来，语言C将继续在工程和技术领域统治，而且，其中光程序的速度是首要标准。因此，这个教规应该作为第一门课程教授给所有工程师作为一种语言，最好是第一语言。换句话说，用别的东西取代C就像替换三角或微积分一样困难！！
